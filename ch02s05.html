⁠2.5. Posibles inconvenientes durante la instalación.

Es poco probable, pero es posible que se produzcan errores durante el proceso de instalación. Algunos de ellos se mencionarán en esta guía, y lo más importante, cómo resolverlos.

Importante

Una vez que la instalación se haya completado al 100%, antes de reiniciar es MUY IMPORTANTE quitar el USB de la computadora o lo reconocerá como parte de la estructura HD y luego fallará.
Error Parsing Kickstart Config
Después de particionar el disco en el proceso de instalación, la partición se llamará VolGroup por defecto. Si realiza una segunda instalación del manager más tarde (sin renombrar el volumen del grupo), obtendrá este error.
Para solucionarlo, es tan simple como eliminar el volgroup. Puede utilizar Gparted para realizar esta acción.
Anaconda Exception Occurred
Este error indica que el nombre del grupo de volúmenes "VolGroup" ya está en uso. Esto se debe a una mala grabación del ISO en el USB de arranque o una mala partición en el HD antes de la instalación.
La solución consiste en volver a crear un usb de arranque asegurándose de que no haya errores. Como se indica en la guía mencionada anteriormente, para grabar la imagen ISO de un sistema Linux en un dispositivo USB, puede usar el comando dd.
