Bienvenido al proceso de instalación de redBorder Enterprise. Gracias a esta guía, cualquiera puede preparar la plataforma sin tener conocimientos específicos de programación en Linux.
Instalar el Manager de la edición de redborder Enterprise es el primer paso para empezar a monitorizar y proteger la información de su red gracias a la solución de redborder. Debe verificar el contenido de este capítulo antes de comenzar la instalación de la plataforma (también conocido como el Manager de redborder).

⁠2.1. Obtención de la ISO

Redborder Enterprise Edition se distribuye como un archivo ISO que se puede obtener de diferentes maneras. Si no tiene el archivo ISO, comuníquese con nuestro equipo de ventas (sales@redborder.com) o con nuestro equipo de soporte (support@redborder.com) para proporcionarle el último archivo ISO disponible o una forma de descargarlo.
