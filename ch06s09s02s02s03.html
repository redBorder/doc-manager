⁠6.9.2.2.3. Configuración DNS y dominio
Al salir del 1) Management Network configuration volvemos a la interfaz del redborder Network configuration menu en el que encontramos la opción de configuración de DNS y dominio. Inicie la configuración seleccionando: 2) DNS and domain settings.
Para realizar la configuración correctamente, el usuario deberá introducir:
  • Nombre de Dominio
  • Dirección DNS primaria
  • Dirección DNS secundaria (opcional)
Una vez introducida esta información, el sistema avisará de que se ha aplicado esta configuración de manera correcta.
Configuración DNS y dominio
Para finalizar el proceso, es necesario aplicar cambios: seleccionar la opción apply insertando la letra a en el menú principal de configuración de red.
