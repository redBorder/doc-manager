⁠2.4. Proceso de Instalación

Tan pronto como su sistema se inicie con la imagen ISO de redborder, verá la pantalla de bienvenida de instalación de redborder Enterprise con opciones útiles relacionadas con el funcionamiento de su computadora...
Pantalla de bienvenida de la instalación de Redborder Enterprise
  • Install Manager (Instalar Manager): Realiza la instalación del software Manager en el equipo.
  • Rescue Installed System (Sistema Instalado de Rescate): Inicia el modo de recuperación en aquellos casos en los que haya problemas para arrancar su máquina (modo avanzado).
  • Boot from Local Drive (Arrancar desde Disco Local): Arranque desde el disco local (predeterminado).
  • Memory Test (Prueba de Memoria): Prueba de memoria disponible para el sistema de instalación.
Selecciona la opción de Instalar Manager para comenzar el proceso de instalación.
El sistema mostrará una pantalla para seleccionar la unidad para instalar la plataforma, la contraseña del usuario root y el modelo de Licencia:
Gracias al asistente de instalación, puede dejar el manager configurado y listo de forma rápida y sencilla sin necesidad de tener habilidades especiales de programación en Linux.
