6.9.2.2. Configuración de red de los sensores Gateway

Una vez ha ejecutado el comando rb_sysconf accederá al Menú de Configuración redborder.
Este menú consta de 4 opciones:
  1. System Configuration:desde esta opción se pueden definir y modificar una serie de ajustes básicos del sistema como el nombre, hora, tipo de teclado, etc.
  2. Network Configuration: esta es la opción desde la cual se realizan todas las configuraciones de red para la correcta implantación de la solución.
  3. Passwords: acceda a esta opción si desea modificar sus contraseñas.
  4. quit: Salir.
Menú de configuración redborder

Nota

Recuerde: selección de opciones predeterminadas: Enter
Como se ha comentado al inicio del capítulo, el primer paso para comenzar la configuración de los sensores Mail Gateway es el establecimiento de los parámetros de red o puesta en red de los sensores.
Para comenzar el proceso seleccionar la opción 2) Network Configuration del menú de Configuración redborder.
Estos son los pasos que hay que seguir para realizar la instalación de los sensores Mail Gateway:
  1. Configuración de la red de gestión:
    Creación del bond y asignación de interfaz de gestión.
  2. Configuración DNS y dominio

Nota

El orden que se siga en los pasos que comprenden la configuración de la red de gestión es indiferente. El orden indicado en este manual de instalación es el que se considera más óptimo.
Hasta que no se haya realizado esta configuración, no se podrá proceder al registro de los sensores en el Manager.

Advertencia

Para la instalación y registro de los sensores Mail Gateway es necesario disponer de tan solo un bond.
En caso de que el sistema se instale en un equipo real que disponga de más de una tarjeta de red, el usuario deberá elegir cuál será el bond de gestión.
6.9.2.2.1. Creación de un nuevo bond y asignación de puertos
Una vez que hemos accedido al redborder Network configuration menu seleccionamos la opción 1) Management Network Configuration desde la cual podremos realizar la creación de un bond para la instalación del sensor.
redborder Network Configuration menu
Para ilustrar este proceso crearemos un bonding (bond0) y le asignaremos 2 puertos.
Seleccione la opción n que aparece en el submenú lateral derecho de la pantalla.
Una vez que haya revisado los puertos que se encuentran disponibles, seleccione aquel que desea asignar al bonding, tal y como aparece en el margen inferior de la pantalla anterior.
Introduzca el número de puerto en la línea Insert bonding first port [0]:
El número que aparece entre corchetes [] es el que define el sistema por defecto. Para seleccionar este número, pulse Enter.
Creación de un bond: asignación de puertos
El puerto seleccionado desaparecerá de la opción de puertos disponibles.

Importante

El sistema solo permite la creación de un máximo de 4 bonds: (0-3).
