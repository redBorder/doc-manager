8.3.3. Hadoop

Hadoop es un framework de procesamiento distribuido de código abierto que gestiona el procesamiento y almacenamiento de datos para aplicaciones de Big Data que se ejecutan en sistemas agrupados.
Redborder utiliza Hadoop para proporcionar análisis de tráfico de Big Data en tiempo real.
Hadoop no está habilitado de forma predeterminada en un administrador Redborder, debido a sus requisitos de recursos. Es muy importante contar con recursos sólidos para ejecutar Redborder con Hadoop, al menos 64 Gb de RAM.
