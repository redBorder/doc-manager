⁠4.9. Opciones

Cada vista tiene una serie de opciones que permiten al usuario, entre otras cosas, mostrar el tráfico total, exportar los datos a un CSV o llevar la máquina a un estado del pasado para analizar y comparar el comportamiento actual y pasado.
A continuación, se enumeran todas las opciones disponibles en esta pestaña. El usuario debe saber que estas opciones varían según la vista seleccionada previamente.
  • Mostrar Total: Al seleccionar esta opción, se muestra en el gráfico el valor de todo el tráfico diferenciado recibido o emitido por cada uno de los atributos que se pueden consultar. Cuando esta opción está activa, aparece la lista de opciones "Ocultar total". Disponible en la vista superior y única.
  • Exportar a CSV: el usuario descargará un archivo en formato CSV que incluye los datos que corresponden al atributo seleccionado, puede establecer un límite de filas para incluir en el archivo. Disponible solo para las vistas Tops y Únicos.
  • Máquina del tiempo: devuelve la máquina a un estado anterior. Al seleccionar esta opción, el usuario puede ver los datos como si los estuviera viendo en el último día y hora especificados. Disponible para todas las vistas excepto para Raw..
  • Guardar pestañas / columnas por defecto: para aplicar esta opción, los datos de columnas o pestañas que se han incluido para el análisis se muestran por defecto en las nuevas consultas realizadas. Disponible para vistas Raw, Únicos y Tops.
  • Ordenar agregaciones: a través de esta opción, el usuario puede elegir el orden en que se muestran las columnas o pestañas. Disponible para la vista Raw.
  • Activar recarga: activa la cuenta regresiva para la recarga de datos. En cualquier momento, el usuario puede detener esto seleccionando Detener la recarga (la opción Iniciar reemplaza la recarga una vez que está habilitada). En el extremo izquierdo de la barra de sección aparece una cuenta regresiva que indica el tiempo de espera para volver a cargar. Disponible para la vista Raw.
