⁠4.5. Vistas: múltiples opciones para la visualización de datos

Las vistas proporcionan múltiples perspectivas para mostrar los datos asociados al tráfico de red.

Nota

Recuerde que en la sección Dashboard puede crear widgets personalizados para cada tipo de aplicación que indique un tipo de vista para mostrar eventos.
Las vistas disponibles para la App de Tráfico/Vault son:
  • Tops: agregación de eventos basada en una unidad para mostrar los más destacados. La suma total de datos de diferentes eventos para mostrarlos como uno.
  • Raw: ver todos los eventos de datos sin procesar, clasificados por atributos y franjas horarias.
  • Comparar: comparar intervalos de tiempo para analizar diferencias en el estado de la red.
  • Único: muestra los diferentes elementos que han interactuado con la red según el atributo.
Vistas

Nota

Las opciones que se muestran en la pestaña Opciones y los atributos Añadir pestaña/columnas variará según el tipo de vista seleccionada.

Importante

Recuerde combinar las vistas con diferentes tipos de gráficos. Esto le dará múltiples posibilidades para el análisis de datos.

⁠4.5.1. Tops

La vista Top nos permite ver cuáles fueron los eventos más destacados a medida que se selecciona el intervalo de tiempo. Lo que imaginamos en el gráfico es la suma de todos estos eventos que se muestran como un solo evento. Si colocamos el ratón sobre cada uno de estos "picos" o Tops, vea el desglose del evento por fecha, hora y tipo de dimensión o atributo que está comprobando.
