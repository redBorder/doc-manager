⁠6.9.2.3.2. Configurar hora local
Seleccionar la opción 2) Set local time.
Insertar fecha y hora de la franja horaria correspondiente al país en el que se encuentre. Esta debería coincidir con la seleccionada en la fase de ajustes básicos durante el proceso de configuración de los sensores IPS.
El sistema le indica dos formatos posibles para la inserción de la fecha y la hora:
  • [HH:mm] ó [YYYY/MM/DD]
  • [YYYY/MM/DD HH:mm]
Pulsar Enter para continuar con el proceso de registro.
