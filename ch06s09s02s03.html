6.9.2.3. Registro de los sensores en el Manager

Para finalizar el proceso de instalación de los sensores Mail Gateway, el último paso que debe llevarse a cabo es el Registro de los sensores en el Manager.
Hasta que no se haya completado el registro, se visualizará un mensaje en el que se indica que los sensores aún no han sido registrados en el Manager.
Inicio del registro de los sensores en el Manager
En el menú de registro de los sensores en el Manager encontramos las siguientes opciones:
  1. Hostname [rbsensor]: asignar un nombre al sensor.
  2. Set local time: configurar hora local.
  3. IP address for rB Master Manager [127.0.0.1]: introducir la IP del Manager.
  4. Register rB Sensor/Manager: registro de los sensores en el manager Sensor/Manager. Hasta que no se haya completado el proceso de registro el usuario visualizará el siguiente mensaje rotulado en rojo: (not registered yet)
6.9.2.3.1. Asignar nombre al sensor (Hostname)
Seleccionar la opción Hostname [rbsensor] del menú de registro de los sensores en el Manager insertando el número 1.
A continuación, en la línea Insert Hostname [rbsensor]: escribir el nombre que se desea dar al sensor que se va a registrar en el Manager. A partir de este momento, este es el nombre que visualizará en la web cuando acceda al listado de sensores IPS.
Asignar nombre al sensor

Nota

Recuerde introducir un nombre coherente para el sensor. Evite utilizar el nombre “Sensor”, ya que ese es el nombre genérico que utiliza el Manager y que a la larga, podría causar confusión.

Advertencia

El sistema le pedirá que verifique que el nombre elegido es el correcto, en ese caso, pulse Enter
Si desea modificarlo inserte la letra n
El sistema procederá a cambiar el nombre del sensor. Para continuar, y regeresar al menú de configuración del sistema redborder, pulse cualquier tecla tal y como se indica Press a key to continue...
