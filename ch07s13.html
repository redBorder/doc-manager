⁠7.13. Versión de reglas

El usuario puede administrar las fuentes de las reglas de la plataforma. Es posible agregar o cargar nuevas fuentes y también habilitar las elegidas. Las reglas se usan para la sonda Snort, que es utilizada por el módulo de tráfico.
Versión de reglas
Es posible cargar reglas para un IPS en particular usando la opción Cargar fuente, donde el usuario puede seleccionar el archivo para cargar.
También es posible agregar fuentes para descargar nuevas reglas de Internet que se actualizan periódicamente Si es necesario, el usuario puede usar la actualización de la regla de fuerza para asegurarse de que todas las reglas estén actualizadas.
