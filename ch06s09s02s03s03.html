⁠6.9.2.3.3. Introducir la IP del rB Master Manager
Seleccionar la opción 3) IP address for rB Master Manager [127.0.0.1].
Insertar la dirección IP o nombre de dominio del equipo utilizado por el usuario. El sistema realizará una serie de comprobaciones para verificar si es alcanzable. Este proceso puede llevar unos segundos.
El sistema sincronizará la hora de los equipos y, a continuación, reiniciará un par de servicios con el objetivo de finalizar la sincronía horaria.
¡Enhorabuena! Su equipo está preparado para realizar el registro de los sensores en el Manager.
