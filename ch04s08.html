⁠4.8. Tipo de gráfico

Como veremos en Capítulo 5, Dashboards puede seleccionar el tipo de gráfico que desea mostrar los datos proporcionados en las diferentes vistas. Las opciones son muy similares a las que están disponibles en el proceso de creación de widgets personalizados..
Los gráficos que están disponibles en todo momento dependerán de la vista seleccionada. Esta opción no aparece, por ejemplo, al aplicar la vista de mapa de los datos, ya que los datos se representan en esta vista, en un mapa. Estas son las posibilidades gráficas de la plataforma Redborder para vistas:
Acumulado: muestra comparativamente en el mismo gráfico los datos para cada uno de los atributos seleccionados. (En la imagen se comparan los distintos sensores de tráfico).
Gráfico Acumulado para vistas de sensor
Área: gráfico que proporciona una superposición de los datos a un grupo de elementos que pertenecen al atributo seleccionado, haciendo que el tráfico total sea la suma del tráfico por atributo (en la imagen, se muestra por cada sensor).
Área para la vista gráfica de Tops por Dirección SRC: múltiples elementos
Si desean mostrar en un solo elemento gráfico, simplemente haga clic en cualquier punto del gráfico para aislar el elemento seleccionado.
Área para la vista gráfica de Tops por Sensor: elemento único
Línea: gráfico que proporciona una superposición de los datos a un grupo de elementos que pertenecen al atributo seleccionado; en este caso, el área no se muestra pero el gráfico está representado por una línea.
Gráfico Top de líneas por sensor
Barras: proporciona una superposición de datos a un grupo de elementos que pertenecen al atributo seleccionado, siendo el total, la suma de valores de cada uno de esos elementos. En este caso, los datos se representan mediante barras.
Gráfico Top de barras por sensor
SBarras: o Barras acumuladas, proporciona en el mismo gráfico los datos obtenidos para cada uno de los elementos incluidos en el atributo seleccionado, para que pueda ver los resultados de IP independiente, sensor, etc., representados en barras.
Gráficos SBarras para vista superior por dirección SRC
Flujos: gráfico solo disponible en la vista Raw. Muestra el flujo de tráfico por sensor.
