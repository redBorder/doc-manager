7.7.2. Loaders

El apartado Loaders (cargadores) está compuesto por dos subapartados:
  • Loaders
  • API Keys
Loaders
En este subapartado se define el peso otorgado a cada cargador de manera que la plataforma pueda realizar una poderación entre todos los cargadores para obtener así el score total.
El objetivo principal de este subapartado es que ecada usuario optimice los valores que aquí se indican según sus necesidades.
Cada componente tiene una serie de cargadores asociados cuyo peso se puede definir en cada una de las pestañas que se muestran: Hash, URL e IP.
Cargadores: Loaders
Los analizadores se reparten en tres secciones en función del tiempo requerido para el análisis:
  • Realtime: cargadores que obtienen información en tiempo real.
  • Fast Batch processing: cargadores de procesado rápido de lotes de información.
  • Slow Batch processing: cargadores de procesado lento de lotes de información.
API Keys
En este apartado el usuario debe introducir las claves que le han sido proporcionadas por los proveedores para la activación de los cargadores.
