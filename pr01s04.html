⁠4. Requisitos para la implantación de redBorder

La plataforma de redBorder se puede instalar mediante dos infraestructuras posibles:
  • Máquina física: disco local de su equipo (Hardware).
  • Máquina virtual: estructura de virtualización o Cloud.

Importante

Requisitos de instalación del Manager de redBorder en un equipo fí­sico:
Memoria y almacenamiento: 60GB de disco duro y un mí­nimo de 16GB de RAM. Siendo recomendables 32GB de RAM.(Esta cifra dependerá del número de sensores asignados).
En caso de que se haya creado un clúster son necesarias dos interfaces de red: una para la red de gestión (Management IP Address) y otra para la red de sincronismo (Sync IP address).

Soluciones Cloud compatibles:

  • VMWare
  • Open Stack
  • Amazon Web Services (AWS)
